# Construction Field Management Web App

## Tech Stack
- React (Vite)
- React Router
- Tailwind CSS

## Features
- Login page with mock authentication
- Project list with card layout
- Daily Progress Report (DPR) form
- Image upload with preview
- Form validation
- Responsive design

## Setup Instructions

Clone the repo:

Step 1: git clone https://github.com/abdullahansari-yk/Construction-Field-Management.git

Step 2: npm install

Step 3: npm run dev
