# Construction Field Management Web App

## Tech Stack
- React (Vite)
- React Router
- Tailwind CSS

## Features
- Login page with mock authentication
- Project list with card layout
- Daily Progress Report (DPR) form
- Image upload with preview
- Form validation
- Responsive design

## Setup Instructions

Clone the repo:

git clone https://github.com/abdullahansari-yk/Construction-Field-Management.git

Install dependencies => npm install

Run the project =>  npm run dev
